### 問題說明

### 解決方案
- ✅ 使用 `--disable-geolocation` 參數禁用地理位置
- ✅ 使用 `--deny-permission-prompts` 拒絕權限提示
- ✅ 覆蓋 `navigator.geolocation` API 提供模擬位置
- ✅ 禁用通知和其他可能的彈窗干擾
